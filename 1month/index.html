Welcome to One Month Study Plan!

Hello programmers! Welcome to one month problem-solving course.
The objective of this course is to make you familiar with python programming language basics and how to use them to solve programming problems. Till the mid of week 2, you will learn fundamentals of programming, later, to build the logic, how to tackle the coding problems which includes mathematical problems, puzzles, string, array, list and matrix related coding problems.
