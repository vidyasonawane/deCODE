Loop Control Statements

Break, Continue, Pass keywords with examples

Break Statement:

  • Break statement brings control out of the loop. It ends the current loop and jumps to the statement immediately following the loop.
  • example:
for letter in 'LearnToCodeWithVidya':
    if letter == 'a':
        break
    else:
        print(letter)
 
print ("Current Letter :", letter)
print ("Program ends")
  • output:
L
e
Current Letter : a
Program ends

Continue Statement:

  • Continue statement returns the control to the beginning of the loop. It ends the current iteration and jumps to the top of the loop and start next iteration.
  • example:
# Print all letters except a and e
for letter in 'LearnToCode':
    if letter == 'a' or letter == 'e':
         continue
    print(letter)
  • output:
L
r
n
T
o
C
o
d

Pass Statement:

  • pass statement is used to write empty loops in python.
  • example:
# An empty loop
for letter in 'LearnToCodeWithVidya':
    pass
print("Last Letter :", letter)
  • Output:
Last Letter : a
